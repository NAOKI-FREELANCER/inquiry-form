## 実装中に問題となったこと
### DBへ値をINSERTする際にDBへの登録ができない
- 原因は、DBの電話番号を登録するカラムに設定していたデータ型（int）が原因でした。
varcharにして解決しましたが根本的なで原因は現時点でわかっていません。
### DBへ登録しても文字化けしてしまう
- 原因はCloud9のMySQLのcharacter_setの初期設定がutf8でないことでした。
utf8に変更することで解決しました。
